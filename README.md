# ScpSpeech

## REQUIREMENTS
* Exiled: 7.0.0
* SCP:SL Server: 13.0

## Commands
| Command                         | Description                                                      |
|---------------------------------|------------------------------------------------------------------|
| .scpsp change                   | Changes the speaker from a ghost to a spectator and vice versa\. |
| .scpsp current                  | See the current voice chat\.                                     |
